# SAProj
Django + Selenium + GPT API를 활용한 상품 리뷰 감성평가 사이트

## 🖥️ 프로젝트 소개
Selenium으로 상품 리뷰를 스크래핑 하고, 해당 리뷰를 GPT API를 활용해 긍정인지, 부정인지 감성 평가하는 사이트 입니다.
